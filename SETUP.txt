STEP 5 (OPTIONAL): ADD REAL API KEY
-----------------------------------
If you want real AI (not demo):

1. Go to: https://console.anthropic.com
2. Sign up (FREE $5 credits)
3. Click "API Keys" in sidebar
4. Click "Create Key"
5. Copy the key (starts with sk-ant-...)

6. Open "script.js" with Notepad/TextEdit
7. Find line 5: const API_KEY = 'YOUR_API_KEY_HERE';
8. Replace with: const API_KEY = 'sk-ant-your-key-here';

9. Find line 6: const USE_DEMO_MODE = true;
10. Change to: const USE_DEMO_MODE = false;

11. Save the file
12. Refresh browser (F5)
13. Done! Now using real AI


QUICK TROUBLESHOOTING
----------------------

Problem: Files not loading?
Solution: Make sure all files are in the SAME FOLDER

Problem: CSS not working?
Solution: Check file is named exactly "styles.css" (not styles.css.txt)

Problem: JavaScript not working?
Solution: Check file is named exactly "script.js" (not script.js.txt)

Problem: API key not working?
Solution: Copy key again, check for extra spaces
